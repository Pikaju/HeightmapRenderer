# Heightmap Renderer

A simple heightmap renderer I wrote when I was 16 years old in C++ and OpenGL.

## Controls

* WASD - Move horizontally
* Space - Move up
* Shift - Move down
* Control - Move faster
* Left mouse button - Grab mouse
* Escape - Release mouse
